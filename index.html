CSS és Bootstrap formázások

Formázások

A weboldalak megjelenésének kialakításában két fontos eszköz áll rendelkezésünkre: a CSS és a Bootstrap. A CSS (Cascading Style Sheets) egy stílusleíró nyelv, amely lehetővé teszi, hogy meghatározzuk, hogyan nézzenek ki a HTML-elemek. Segítségével beállíthatjuk például a színt, a betűméretet, a margókat, a térközöket, az igazítást vagy akár az animációkat is. A CSS teljes szabadságot ad a fejlesztő kezébe: minden egyes elem megjelenését egyedileg lehet formázni, így teljesen testreszabott, egyedi dizájnt hozhatunk létre. Ugyanakkor ez időigényes lehet, hiszen minden stílust nekünk kell megírnunk.

Ezzel szemben a Bootstrap egy előre elkészített CSS és JavaScript alapú keretrendszer, amelyet azért hoztak létre, hogy felgyorsítsa és leegyszerűsítse a webfejlesztést. A Bootstrap előre definiált osztályokat kínál, amelyeket egyszerűen hozzáadhatunk HTML-elemekhez, így azok automatikusan modern, reszponzív megjelenést kapnak. Például egy gomb, egy űrlapmező vagy egy rácselrendezés néhány osztály hozzáadásával máris jól néz ki és mobilbarát lesz. A Bootstrap különösen hasznos akkor, ha gyorsan szeretnénk egységes, professzionális kinézetű weboldalt készíteni anélkül, hogy minden stílust nulláról kellene megírnunk.
